# Prompt 提示词目录

本目录包含 Academic Web 项目的完整部署提示词，可用于 AI 辅助部署或手动配置。

## 文件列表

### DEPLOY_PROMPT.md

**主部署提示词**

详细的部署指南，包含：
- 项目初始化步骤
- 前端技术栈配置
- 页面组件说明
- API 服务配置
- Edge Functions 实现
- KV Storage 配置
- 环境变量设置
- 功能清单
- 部署检查清单

**使用方法：**
1. 打开 EdgeOne 控制台或使用 EdgeOne CLI
2. 按照提示词内容逐步配置
3. 参考检查清单验证部署

### SKILLS_GUIDE.md

**Skills 使用指南**

介绍如何使用 `Academic-Web-skills/` 中的技能包，包含：
- 技能包目录结构
- 各技能功能说明
- 使用方法
- 前提条件

## 使用示例

### 使用 AI 助手部署

将 `DEPLOY_PROMPT.md` 的内容复制给 AI 助手，例如：

```
请帮我部署 Academic Web 项目，按照以下提示词配置：
[粘贴 DEPLOY_PROMPT.md 的内容]
```

### 手动参考

按顺序阅读：
1. `SKILLS_GUIDE.md` - 了解可用的技能
2. `DEPLOY_PROMPT.md` - 详细部署步骤
